# 1차 개선 재측정 요약 (2026-04-07)

이 문서는 2차 개선 적용 이전에 남아 있던 1차 개선 재측정 결과의 요약 수치를 보존하기 위한 기록이다.

현재 저장소에는 1차 시점의 완전한 raw 리포트나 라운드별 stability 스냅샷이 따로 남아 있지 않다. 대신 이전 재측정 결과 문서에 남아 있던 대표 수치를 현재 구조에 맞춰 옮겨 적었다.

## 기록 범위

- 이 문서는 대표 요약 수치만 보존한다.
- 라운드별 p95, stability status, raw latest 리포트 경로는 따로 남아 있지 않다.
- 따라서 1차 개선 수치는 `중간 결과 비교용 요약 기록`으로 읽는 편이 맞다.

## 대표 수치

- 메시지 히스토리 최신 페이지 p95 median: `39.699ms`
